API Access & Integration

Automate code forensic checks directly within your own infrastructure.

What is the CodeGuard API?

The CodeGuard AI API allows developers, schools, and businesses to integrate our advanced plagiarism and AI-generation detection engine directly into their own systems. Instead of manually uploading files through our web dashboard, your servers can talk to our servers programmatically, enabling seamless automation at scale.

Primary Use Cases

Automated CI/CD Pipelines

Integrate CodeGuard directly into your GitHub Actions, GitLab CI, or Jenkins pipelines. Automatically scan every pull request. If the structural similarity or AI-generation score exceeds your company's threshold, the PR is automatically blocked from merging, keeping your codebase pure.

LMS Integration for Education

Universities can connect their Learning Management Systems (Canvas, Moodle, Blackboard) to our API via webhooks. The moment a student submits a programming assignment, CodeGuard automatically processes the AST logic in the background and attaches a forensic similarity report to the teacher's grading view.

Bulk Repository Scanning

Enterprise clients undertaking massive security audits or legacy code ingestion can use the API to batch-submit thousands of files concurrently, bypassing frontend limitations to map out code origin dependencies across enormous monolithic repositories.

Developer Quickstart

A simple REST endpoint to submit code for analysis. Generate your API key in the dashboard.

cURL Example

curl -X POST http://api.codeguard.ai/v1/scan \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filename": "auth_controller.js",
    "code": "const login = (req, res) => { ... }"
  }'

Node.js Example

const axios = require('axios');

async function scanCode(codeString) {
    const response = await axios.post('https://api.codeguard.ai/v1/scan', {
        filename: 'script.py',
        code: codeString
    }, {
        headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
    });
    
    console.log('AI Score:', response.data.ai_score);
    console.log('Plagiarism Probability:', response.data.plagiarism_score);
}
